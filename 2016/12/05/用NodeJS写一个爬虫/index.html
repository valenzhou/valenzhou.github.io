简介

用NodeJS下一个爬虫,用来获取简书首页的文章,然后保存以txt文件的形式保存在本地,数量为20篇.

初始化项目

​ npm init || yarn init

安装依赖

​ npm install superagent cheerio —save || yarn add superagent cheerio
